## 二、网络模型设计（ColorNet）
    针对颜色分类任务设计的轻量级卷积神经网络。

    设计思路：
    1. 颜色属于低层视觉特征，不需要极深的网络即可有效提取。
    2. 第一层使用 5x5 大卷积核，扩大感受野，更好地捕获颜色区域信息。
    3. 后续层使用 3x3 标准卷积核，逐步提取更抽象的特征。
    4. 全局平均池化(GAP)替代传统全连接前的 Flatten,大幅减少参数量，降低过拟合风险。
    5. BatchNorm 加速收敛,Dropout 增强泛化能力。

    层数统计(仅统计带可训练参数的层,不含激活函数、BN、池化、Dropout);
    - 卷积层:4 层
    - 全连接层:2 层
    - 总计:6 层（符合 <= 10 层、FC <= 3 层的要求）
### 每一层卷积层尺寸变化流程

```
输入图像:        [B, 3, 64, 64]
  ↓
Conv1 (5×5)      [B, 32, 64, 64]
BatchNorm1       [B, 32, 64, 64]
ReLU             [B, 32, 64, 64]
MaxPool (2×2)    [B, 32, 32, 32]
  ↓
Conv2 (3×3)      [B, 64, 32, 32]
BatchNorm2       [B, 64, 32, 32]
ReLU             [B, 64, 32, 32]
MaxPool (2×2)    [B, 64, 16, 16]
  ↓
Conv3 (3×3)      [B, 128, 16, 16]
BatchNorm3       [B, 128, 16, 16]
ReLU             [B, 128, 16, 16]
MaxPool (2×2)    [B, 128, 8, 8]
  ↓
Conv4 (3×3)      [B, 256, 8, 8]
BatchNorm4       [B, 256, 8, 8]
ReLU             [B, 256, 8, 8]
MaxPool (2×2)    [B, 256, 4, 4]
  ↓
GlobalAvgPool    [B, 256, 1, 1]
Flatten          [B, 256]
Dropout(0.5)     [B, 256]
  ↓
FC1              [B, 128]
BatchNorm_FC     [B, 128]
ReLU             [B, 128]
  ↓
FC2 (输出)        [B, 3]
```
## 训练模型

**运行代码为`train.py`**

**训练流程说明：**

1. **数据加载**：从 `dataset/train` 加载训练数据，自动按文件夹名划分为 3 个类别。
2. **预处理与增强**：训练图片经过 Resize、翻转、旋转、颜色抖动等增强后进入网络。
3. **前向传播**：数据通过 ColorNet，输出 3 维 logits。
4. **计算损失**：使用带权重的交叉熵损失（含标签平滑）。
5. **反向传播**：AdamW 优化器更新权重。
6. **学习率调整**：每个 epoch 后，CosineAnnealingLR 更新学习率。
7. **评估保存**：每 10 个 epoch 评估 test1/test2，保存最佳模型到 `pth/` 目录。

## 三、测试模型

代码文件为`test.py`
**测试流程说明：**

1. **加载模型**：从 `pth/` 目录加载最佳模型权重。
2. **遍历测试集**：分别对 `test1` 和 `test2` 进行前向推理。
3. **TTA 集成**：每张图片经过原图+两次翻转，取平均 logits。
4. **统计输出**：
   - 打印**整体正确率**
   - 打印**每一类的正确率**（正确数/总数）

## 预期效果
在 200 个 epoch 内，通过上述优化策略，模型在 `test1` 能达到 **99.54%**的整体正确率,对蓝色球的识别正确率达到**100.00%**，对红色球的识别正确率达到**98.77%**，对黄色球的识别正确率达到**100.00%**。
在 200 个 epoch 内，通过上述优化策略，模型在 `test2` 能达到 **97.14%**的整体正确率,对蓝色球的识别正确率达到**100.00%**，对红色球的识别正确率达到**93.88%**，对黄色球的识别正确率达到**97.56%**。
